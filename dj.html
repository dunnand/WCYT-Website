DJ Panel

⚠ Setup Required

To use the DJ Panel, complete the one-time Google Apps Script setup:

  1. Go to script.google.com → New project
  2. Paste in the contents of dj-backend.gs (in this site's repo) and save
  3. Run ▶ the authorize function once and approve the permissions popup
  4. Deploy → New deployment → Web app → Execute as Me, access Anyone
  5. Copy the Web app URL (ends in /exec)
  6. Paste it into SCRIPT_URL in dj.html and DJPANEL_URL in playlist-widget.js
