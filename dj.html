DJ Panel

⚠ Setup Required

To use the DJ Panel, complete the one-time JSONBin setup:

  1. Create a free account at jsonbin.io
  2. Create a new bin with this content:
    {"wcyt":{"active":false},"2pt0":{"active":false}}
  3. Set the bin to Public (so the website can read it)
  4. Copy your Bin ID and API Key
  5. Edit dj.html and fill in BIN_ID and WRITE_KEY in the config section
  6. Also update playlist-widget.js with the same BIN_ID
